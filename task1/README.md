# Задание 1

## Задача

Заполнить масив float/double значениями синуса (один период на всю длину массива), количество элементов 10^7. Посчитать сумму и вывести в терминал.

## Сборка программы

В директории `/task1`:
```bash
cmake -S . -B build -DUSE_DOUBLE=ON   # double
cmake -S . -B build -DUSE_DOUBLE=OFF  # float
```

При неуказании `USE_DOUBLE` по-умолчанию используется тип `double`.

## Вывод программы

Ожидаемый вывод суммы значений синуса на периоде - 0.0.

* Вывод с типом `float`: 0.094364
* Вывод с типом `double`: 4.89582e-11
